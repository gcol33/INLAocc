# Plot diagnostics for JSDM

Plot diagnostics for JSDM

## Usage

``` r
# S3 method for class 'occu_inla_jsdm'
plot(x, which = 1:2, ...)
```

## Arguments

- x:

  fitted occu_inla_jsdm object

- which:

  integer vector: which plots to show. 1 = species occupancy, 2 = factor
  loadings heatmap.

- ...:

  additional arguments passed to plot

## Value

The object `x`, returned invisibly.
