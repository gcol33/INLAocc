# Plot predicted effects from an occupancy model

Plot predicted effects from an occupancy model

## Usage

``` r
# S3 method for class 'occu_prediction'
plot(x, ...)
```

## Arguments

- x:

  an `occu_prediction` object from `predict(fit, terms = ...)`

- ...:

  additional arguments passed to
  [`plot`](https://rdrr.io/r/graphics/plot.default.html)

## Value

The input object, invisibly. Called for side effect of producing a plot.
