# Plot diagnostics for temporal occupancy model

Plot diagnostics for temporal occupancy model

## Usage

``` r
# S3 method for class 'occu_inla_temporal'
plot(x, which = 1:2, ...)
```

## Arguments

- x:

  fitted occu_inla_temporal object

- which:

  integer vector: which plots to show. 1 = occupancy over time, 2 =
  detection over time.

- ...:

  additional arguments passed to plot

## Value

The object `x`, returned invisibly.
