# Predict from a JSDM

Returns per-species occupancy predictions (no detection process).

## Usage

``` r
# S3 method for class 'occu_inla_jsdm'
predict(object, ...)
```

## Arguments

- object:

  fitted occu_inla_jsdm object

- ...:

  additional arguments (ignored)

## Value

named list of per-species occupancy probability vectors
