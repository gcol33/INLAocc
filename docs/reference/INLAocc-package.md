# INLAocc: Occupancy Models via Integrated Nested Laplace Approximation

Fits single-species, multi-species, spatial, temporal, integrated, and
spatially-varying coefficient occupancy models using Integrated Nested
Laplace Approximations (INLA). Provides a fast alternative to MCMC-based
frameworks like 'spOccupancy', with a compatible API including
mixed-model random effects syntax, posterior predictive checks, WAIC,
k-fold cross-validation, and design-matrix prediction. Uses an EM
algorithm with INLA at each M-step for the two-process
(occupancy/detection) likelihood.
