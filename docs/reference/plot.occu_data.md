# Plot detection history patterns

Produces a 2x2 panel: detection frequency histogram, per-visit detection
rates, visit completeness, and a spatial detection map (if coordinates
are available).

## Usage

``` r
# S3 method for class 'occu_data'
plot(x, ...)
```

## Arguments

- x:

  an `occu_data` object

- ...:

  ignored

## Value

Invisible `NULL`.
